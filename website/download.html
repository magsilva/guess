Download

If you would just like to run GUESS you have the option:

  • Version 1.0-beta Install the latest beta (8/19/2005). This does not come with an installer, but has many more features.
  • Version 0.5-alpha Available in ZIP or EXE (windows self insalling executable)
  • All releases

I am working on a general installer for all systems, but please let me know if you have any problems installing in the meantime. In any case you will need:

  • The Java Runtime (1.4+)
  • The Java Media Framework (If you want to save out animations, should work without this, but I make no promises)

You will want to read the README.TXT and tutorial/walk through in the manual.

Source

GUESS is distributed under a GPL license. The source is included in the guess-src.jar file which is included in all the beta distributions. The current development version is also available through anonymous CVS on SourceForge.
